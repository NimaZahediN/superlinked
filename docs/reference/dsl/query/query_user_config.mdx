## QueryUserConfig

```python
QueryUserConfig(with_metadata=False, redis_hybrid_policy=None, redis_batch_size=250)
```

</ParamField>

</ParamField>

<ParamField path="redis_batch_size" type="int | None" default="250">
The batch size for Redis operations.
</ParamField>

### Properties

<ParamField path="redis_batch_size" type="int | None">

</ParamField>

</ParamField>
