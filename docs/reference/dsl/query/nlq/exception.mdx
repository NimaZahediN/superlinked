## NLQException

```python
NLQException(*args, **kwargs)
```

Common base class for all non-exit exceptions related to natural language query processing.

### Inheritance

**Inheritance Chain**: 
- `NLQException` 
- → `InvalidInputException`
- → `Exception`
- → `BaseException`
