## DuplicateElementException

```python
DuplicateElementException(*args, **kwargs)
```

Common base class for all non-exit exceptions related to duplicate elements in the registry.

### Inheritance

**Inheritance Chain**: 
- `DuplicateElementException` 
- → `Exception` 
- → `BaseException`
